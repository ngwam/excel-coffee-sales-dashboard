Coffee Sales Dashboard (Excel)

Project Overview
This project is an interactive Coffee Sales Dashboard built in Microsoft Excel to analyse sales performance, customer behaviour, and product trends. The project demonstrates advanced Excel techniques for data preparation, transformation, analysis, and visualisation.

Project Workflow

The project followed a complete data analysis process:

- Combined data from multiple tables.
- Performed data transformation using advanced Excel formulas.
- Created an analysis-ready dataset.
- Built Pivot Tables and Pivot Charts.
- Designed an interactive dashboard with slicers and timelines.

Data Sources
The project initially contained three separate tables:

- Orders
- Customers
- Products

The Orders table was enriched by retrieving customer and product information from the other tables using lookup functions.

1. Advanced Excel Functions Used
XLOOKUP

Used to retrieve customer and product information from separate tables into the Orders table.

2. INDEX + MATCH

Used as an alternative lookup technique for flexible and dynamic data retrieval.

3.Multiple Nested IF Statements

Used to classify records based on business rules.

4.IF Function

Applied to create conditional logic and calculated fields.

5. Date Functions

Used to derive Year, Month and Quarter for time-based analysis.

6.Text Functions

Used where necessary to clean and standardise data.

7.Data Validation

Applied to improve data quality.

8.Named Ranges

Used to improve formula readability and workbook management.
